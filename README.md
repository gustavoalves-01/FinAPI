# FinAPI

This is a simple API developped with Node JS only for study purpouses, with the main objective as improve and consolidate the NodeJS + Express fundamentals.

## Features
- Create account;
- Update account;
- Delete account;
- Make a deposit;
- Make a withdraw;
- Get statement;
- Get statement by date;
- Get account balance;

#### Time to Develop: 1.5h
